Welcome to :code:`hyperparameter-tuning`'s documentation!
=========================================================

The :code:`hyperparameter-tuning` package is a minimal and easy-to-use 
hyperparameter tuning framework to help you train hundreds of models.
It's essentially a set of helpful wrappers over `optuna`_.

Full code available on the `GitHub repository`_.
